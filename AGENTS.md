# POD 商品图批量处理系统

## 项目目标

开发一个内部使用的 POD 自动化后台，帮助用户完成图片上传、批量处理、套图、AI生成上架信息、导出 Excel 和图片 ZIP。

## 技术栈

- Next.js
- TypeScript
- Tailwind CSS
- Supabase
- Sharp
- 后续支持豆包和千问 API

## 第一版范围

1. 图片批量上传
2. 素材库
3. 批量改尺寸
4. 批量清晰化接口预留
5. 批量抠图接口预留
6. 固定商品套图
7. AI生成标题、描述、标签、SKU
8. 商品草稿
9. 导出 Excel
10. 导出图片 ZIP
11. 任务中心

## 暂不实现

1. 自动爬虫找图
2. 完整 SDS 自动绑定
3. 多平台自动发布
4. 支付功能
5. 复杂团队权限

## 开发规则

1. 每次只完成用户指定任务，不要自行扩展大功能
2. 所有 API Key 必须放在环境变量，不能写死在代码里
3. 不能提交 .env 文件
4. 前端不能暴露豆包、千问、SDS 的 API Key
5. 所有 AI 调用必须走后端接口
6. 所有批量任务必须记录成功数、失败数、失败原因
7. 所有图片处理结果必须保留原图记录
8. 每次改动后必须运行 npm run lint 和 npm run build
9. 新增功能必须更新 README
10. 如果不确定业务逻辑，先问我，不要自己猜
