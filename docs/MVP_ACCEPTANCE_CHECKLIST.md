# POD 商品图批量处理系统 MVP 验收清单

## 使用说明

- 验收前先完成 `.env.local` 配置，并确认 Supabase migration 已执行。
- 建议按本清单从上到下验收，先准备测试图片，再验证上传、处理、套图、商品草稿和导出。
- “通过/失败记录”栏可填写：通过、失败、阻塞、待复测。
- “备注”栏记录实际问题、失败截图路径、错误信息、数据 ID 或复测结论。

## 测试数据建议

| 测试数据 | 准备要求 | 用途 |
| --- | --- | --- |
| 10 张普通印花图 | jpg、png、webp 混合，尺寸适中 | 上传、批量改尺寸、批量套图、草稿和导出 |
| 50 张普通印花图 | 文件名尽量不同，格式混合 | 上传稳定性和列表展示 |
| 超大图片 | 单张高分辨率大文件 | 上传、元数据读取和失败提示 |
| T 恤底图 | 至少 1 张 2000x2000 商品底图 | 创建 T 恤套图模板 |
| 固定详情图 | 不需要叠加印花的底图 | 验证 `need_print = false` 场景 |

| --- | --- | --- | --- | --- |
